# React SPA + Java GQL + Fargate example

## Backend
* found in javaqldemo/
* run `./gradlew bootRun`

## Frontend
* found in `./webapp/`
* run yarn start to start

## Infrastruction
* in `./infra/`
* if you haven't yet deployed run `yarn cdk bootstrap`
* to deploy run `yarn cdk deploy <StackName>`
